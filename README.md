# 🔵 Vibe — Employee Experience Platform

A full-stack employee engagement application built in Python and Streamlit, powered by synthetic data engineered to simulate real organisational patterns across 9 departments and 500 employees.

---

## What is Vibe?
Vibe is an AI-enabled employee experience platform that gives HR leaders, managers, and employees real-time visibility into engagement, wellbeing, recognition, and participation — all in one place. Built as a portfolio proof-of-concept to demonstrate people analytics thinking, HR product design, and technical execution at CHRO level.

---


## Tech stack

| Layer | Technology |
|---|---|
| Framework | Streamlit |
| Data | Pandas, NumPy |
| Charts | Plotly Express, Plotly Graph Objects |
| Sentiment | VADER |
| Data | Synthetic — 500 employees, 52 weeks, 9 departments |
| Deployment | Streamlit Cloud / Railway |
